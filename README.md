# Agile Login System

## Objetivo
Sistema simples de login e gerenciamento de tarefas desenvolvido para a disciplina de Engenharia de Software.

## Funcionalidades
- Login de usuário
- Adicionar tarefas
- Visualizar tarefas

## Tecnologias utilizadas
- Python
- GitHub
- GitHub Actions

## Metodologia
Foi utilizada metodologia Kanban para organização das tarefas.


## Mudança de Escopo

Foi adicionada uma melhoria no sistema para preparação futura de priorização de tarefas.
